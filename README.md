# 🏟️ StadiumSync Pro

Ultra-lightweight platform for real-time stadium crowd management and express services.

## 🚀 Quick Setup
1. **Install:** `pip install -r requirements.txt`
2. **Run:** `python -m app.main`
3. **Deploy:** `gcloud run deploy --source .`

## 🛠️ Tech Stack
- **Backend:** FastAPI (Python)
- **Real-time:** WebSockets
- **Database:** SQLite
- **Frontend:** Vanilla JS + Tailwind CSS
